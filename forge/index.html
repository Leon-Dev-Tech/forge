WeChat Image Long-press Solution

This page demonstrates why images inside Shadow DOM cannot be long-pressed in WeChat and provides a micro-frontend (MFE) safe solution.

✅ Recommended: MFE Safe Bridge (globalThis)

Status: Working. Bypasses Shadow DOM and window restrictions. Click the image to trigger native WeChat preview.

❌ The Problem: Standard Shadow DOM

Status: Broken. WeChat cannot "see" this image for long-press menus because it is inside a Shadow Root.

❌ Deep Problem: Nested Shadow DOM

Status: Broken. Even if the child is Light DOM, a Shadow DOM ancestor will block WeChat's recognition.

Status: Working. Standard Stencil component with shadow: false.
